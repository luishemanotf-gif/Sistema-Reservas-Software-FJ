# Sistema-Reservas-Software-FJ

Este proyecto implementa un sistema de gestión de clientes, servicios y reservas para la empresa Software FJ.

El sistema fue desarrollado utilizando programación orientada a objetos en Python, aplicando conceptos como:

- Abstracción
- Herencia
- Polimorfismo
- Encapsulación
- Manejo de excepciones

## Estructura del proyecto

- main.py → Ejecuta el sistema y simula las operaciones
- cliente.py → Clase Cliente con validaciones
- servicio.py → Clase abstracta Servicio
- servicios_especializados.py → Servicios derivados (reservas de salas, alquiler de equipos y asesorías)
- reserva.py → Gestión de reservas
- excepciones.py → Excepciones personalizadas
- logger.py → Registro de errores
- logs/errores.log → Archivo donde se guardan los errores del sistema

## Funcionalidades

- Registro de clientes
- Gestión de servicios
- Creación y cancelación de reservas
- Manejo de errores con excepciones
- Registro de eventos y errores en archivo de logs
